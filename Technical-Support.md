## 二、常见问题 FAQ
### 1. 我的内容 / 数据丢失了，怎么找回？
ClipMemo 为纯本地离线应用，我们不运营任何服务器，无法访问、存储您的任何数据。
若您未开启 iCloud 同步：数据仅存储在您的设备本地，卸载 App、设备格式化、系统还原会导致本地数据永久删除，无法恢复，请您定期手动备份重要内容。
若您已开启 iCloud 同步：可在 App 内手动触发同步，或前往 iOS 系统「设置 - Apple ID-iCloud」，确认已开启 ClipMemo 的 iCloud 权限，等待系统同步完成后即可恢复数据。
### 2. 面容 ID 解锁无法开启 / 失效怎么办？
请先确认您的设备支持面容 ID 功能，且已在 iOS 系统「设置 - 面容 ID 与密码」中完成面容录入。
前往 iOS 系统「设置 - ClipMemo」，确认已开启面容 ID 权限。
若仍无法使用，可重启 App 或设备后重试，该功能为系统级能力，我们无法获取、修改您的任何生物特征信息。
### 3. 应用自带键盘无法使用 / 不显示怎么办？
前往 iOS 系统「设置 - 通用 - 键盘 - 键盘」，确认已添加 ClipMemo 键盘。
前往 iOS 系统「设置 - ClipMemo」，确认已开启键盘相关权限。
若仍无法使用，可重启 App 或设备后重试，键盘所有数据仅在本地处理，不会上传任何内容。
### 4. 购买付费功能后，无法解锁权益 / 恢复购买失败怎么办？
请确认您使用的 Apple ID，与购买时的 Apple ID 为同一个账号。
确保设备网络正常，前往 App 内的付费页面，点击「恢复购买」按钮，系统会自动同步您的购买状态。
所有交易均由苹果官方 App Store 处理，我们无法获取、修改您的订单信息，若仍有问题，可联系苹果官方客服核实订单状态。
### 5. 怎么取消订阅 / 关闭自动续费？
所有订阅服务均由苹果官方管理，您可通过以下方式操作：
打开 iOS 系统「设置」，点击顶部的 Apple ID
选择「订阅」，找到 ClipMemo 对应的订阅项目
点击「取消订阅」，按系统提示完成操作即可
取消订阅后，您仍可使用付费权益至当前订阅周期结束。
### 6. 卸载 App 后，数据能恢复吗？
未开启 iCloud 同步：卸载 App 会同时删除所有本地数据，无法恢复。
已开启 iCloud 同步：重新安装 App 后，开启 iCloud 权限，等待系统同步完成即可恢复数据。
### 7. iCloud 同步不生效怎么办？
确认您的 Apple ID 已正常登录 iCloud，且设备网络正常。
前往 iOS 系统「设置 - Apple ID-iCloud」，确认已开启 iCloud 云盘，且已开启 ClipMemo 的同步权限。
确认您的 iCloud 存储空间充足，存储空间不足会导致同步失败。
### 8. 如何开启和使用连续复制功能？
连续复制功能需先开启键盘的「允许完全访问」权限，具体操作与使用步骤如下：
前往 iOS 系统「设置 - 通用 - 键盘 - 键盘 - ClipMemo」，开启「允许完全访问」开关；
调出 ClipMemo 键盘，点击键盘左上角的菜单按钮，选择「连续复制」选项；
进入连续复制页面后，该功能即自动开启，您可正常使用连续复制相关能力。
* 注：即使开启「允许完全访问」权限，ClipMemo 键盘仍不会记录、上传、收集您的任何输入内容与个人信息，所有数据仅在您的设备本地处理，严格遵循我们的隐私政策。
### 9. 如何使用 “分词选择” 功能？
调出 ClipMemo 键盘，切换至「剪贴板」列表页面；
长按您需要处理的剪贴板内容卡片，会自动弹出分词选择菜单；
点击菜单中的「分词选择」选项，即可启用该功能，对内容进行分词处理与选择操作。
